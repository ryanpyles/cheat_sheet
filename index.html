HTML Tables Cheat Sheet

Tag/Attribute Example Description
<table>
Example Table
Defines a table.
<tr>
Table Row
Defines a row within a table.
<td>
Table Data
Defines a cell within a table row.
<caption>
Table Caption
Defines a caption for the table.
<thead>, <tbody>, <tfoot>
Header
Body Content
Footer Content
Defines sections within the table: header, body, and footer.
Structure <table> <thead> <tr> <th>Header</th> </tr> </thead> <tbody> <tr> <td>Data</td> </tr> </tbody> <tfoot> <tr> <td>Footer</td> </tr> </tfoot> </table> Illustrates the structure of a table using thead, tbody, and tfoot sections.
Layout <table> <tr> <th>Header 1</th> <th>Header 2</th> </tr> <tr> <td>Data 1</td> <td>Data 2</td> </tr> </table> Explains how to structure a basic table layout with headers and data cells.

CSS Selectors and Specificity

Selector Example Description
Universal Selector * {...} Selects all elements on the page.
Element Selector div {...} Selects all <div> elements.
Class Selector .my-class {...} Selects elements with class .my-class.
ID Selector #my-id {...} Selects an element with ID #my-id.
Descendant Selector div p {...} Selects <p> elements that are descendants of <div> elements.

Common CSS Properties

Property Example Description
color color: blue; Sets the text color.
font-family font-family: 'Arial', sans-serif; Sets the font family for text.
font-size font-size: 16px; Sets the font size.
font-weight font-weight: bold; Sets the font weight.
text-align text-align: center; Sets the alignment of text.
background-color background-color: #ffcc00; Sets the background color of an element.
margin margin: 10px; Sets the margin space around an element.
padding padding: 10px; Sets the padding space within an element.
border border: 1px solid #ccc; Sets the border style around an element.
width width: 200px; Sets the width of an element.
height height: 150px; Sets the height of an element.
display display: inline-block; Sets how an element should be displayed.
float float: left; Sets the floating behavior of an element.
position position: relative; Sets the positioning of an element.
z-index z-index: 1; Sets the stacking order of an element.
